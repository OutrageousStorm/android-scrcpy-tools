# 🖥️ scrcpy Tools

Scripts and launchers for [scrcpy](https://github.com/Genymobile/scrcpy) — the best Android screen mirror tool.

## Scripts

| Script | What it does |
|--------|-------------|
| `launch.py` | Interactive scrcpy launcher with profiles |
| `record.py` | Record screen with auto-named output files |
| `automate.py` | Replay tap/swipe sequences from a script file |
| `multi_device.sh` | Mirror multiple devices side-by-side |

## Install scrcpy
```bash
# Mac
brew install scrcpy

# Linux
sudo apt install scrcpy

# Windows
# Download: https://github.com/Genymobile/scrcpy/releases
```
